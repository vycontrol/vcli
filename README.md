# vcli

## config files

those files like .aws/ files

```
.vcli/
.vcli/config
.vcli/auth
```

### .vcli/config
global configurations

### .vcli/auth
users configurations

```
[email]
apikey=APIKEY
```


## register
usage

vcli register [alias] [--email email]

- **alias** if not provided alias will generate an alias from random names (like docker containers names)
- **email**
